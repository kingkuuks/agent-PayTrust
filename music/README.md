# Background Music

Place royalty-free MP3 tracks here. The agent maps them to visual styles:

| Visual Style | Track |
|--------------|-------|
| dark_urgent | On The Flip - The Grey Room & Density & Time.mp3 |
| clean_professional | For Our Friends - Telecasted.mp3 |
| warm_community | From Here on In - Everet Almond.mp3 |
| default | Call me crazy - Patrick Patrikios.mp3 |

To change the mapping, edit `BGM_STYLE_MAP` in `agents/asset_builder.py`.
